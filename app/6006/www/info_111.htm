昭泰传媒招聘

发布时间：2010-11-27 22:58:10　　信息来源：曲江人才网　　访问次数：1139

主编

岗位职责

1. 根据《青锐·Future》杂志的品牌定位和发展方向，确定编辑方针，完善、管控杂志的编辑出版流程，保证杂志的编校质量和印刷质量；

2. 随时关注市场需求、行业动态及热点话题，定期组织实施读者调查；

3. 领导团队根据杂志定位，创新性的完成每期内容的选题策划及内容编制；

4. 协同市场部进行相关推广工作；

5. 制定内容资源合作计划，统筹杂志合作单位；

6. 管理和激励团队，营造坦诚沟通、鼓励创新的团队氛围，重视员工成长。

任职资格：

1. 新闻、教育、中文、心理学、人力资源管理、哲学等相关专业本科以上学历；

 2. 五年以上媒体工作经验，其中三年以上同等岗位工作经验，熟悉杂志编辑出版流程；

3. 具有较强的编辑、组织、策划、创新能力，文字功底深厚；

4. 热爱媒体事业，对于帮助大学生解决成长和就业问题具有使命感；

5. 开放、务实，具备优秀的团队建设能力。

文字编辑

岗位职责：

1. 开展杂志选题项目及内容的策划、实施；

2. 独立完成稿件的初审、编辑加工、校对等工作；

3. 组织编辑活动，安排排版并对质量和进度负责；

4. 收集并及时处理作者、读者意见和反馈信息；

5. 研究有关学术动态和编辑出版信息，提出选题和组稿设想。

6. 建立并拓展维护外编资源。

任职资格：

1. 人力资源管理、心理学、新闻、教育等相关专业本科以上学历，有杂志编辑工作经验更佳；

2. 具有较强的编辑、组织、策划、创新能力，具有较强的文字功底和逻辑分析能力；

3. 新闻触觉敏锐，能够及时收集业界信息，能组织、策划和完成专题；

4. 工作细致耐心，责任心强，具有良好的沟通协调能力，具有团队合作精神。

美术编辑

岗位职责：
       1. 负责《青锐·FUTURE》杂志的视觉效果设计、杂志排版；
       2. 负责公司各类宣传品的设计制作；
       3. 负责客户平面广告的设计制作。
       任职资格：
       1. 美术、艺术设计、广告设计相关专业，本科学历。
       2. 一年以上杂志美术编辑工作经验，熟悉杂志版式设计制作，了解杂志采编、出版、印刷工作流程。
       3. 精通平面设计软件Photoshop、Illustrator、Indesign、Pagemaker、Coredraw等，懂网页设计者优先。
       4. 良好的美术功底及艺术直觉，思维开阔，创意独特，关注国际设计潮流。
       5. 善于沟通，学习能力强，良好的团队合作精神，较强的执行能力，能承受一定的工作强度和工作压力。

广告客户经理

岗位职责：
       1.负责公司杂志《青锐·future》的广告业务销售；
       2.在销售合同履行过程中，负责与客户的联络、协调工作；
       3.负责尾款回收工作，确保销售收入指标的达成。

       任职资格：
       1.教育背景：本科以上学历，特别优秀者可放宽学历要求。
       2.工作经验：有两年以上广告销售从业经验，在以往的工作岗位上无不良工作记录。对于特别优秀的应届毕业生可放宽工作经验要求。
       3.技能技巧：沟通能力强，善于把握客户需求，学习能力强。
       4.工作态度：认真负责，以结果为导向，高度的工作热情，良好的团队合作精神，较强的执行能力，遵守公司规章制度，坚守商业道德，能承受一定的工作强度和工作压力。

5.从事过广告行业者优先。

人力资源总监

工作职责：

1.全面统筹规划公司的人力资源战略；

2.建立并完善人力资源管理体系、制度、流程（包含招聘、绩效、培训、薪酬及员工发展等体系的全面建设），监督公司人力资源相关制度的实施；

3.根据公司招聘流程，协调公司总部及全国各地办事处员工招聘工作，负责所有应聘员工的薪酬谈判，实施人才发展储备计划；

4.根据企业的发展目标，组织、实施培训工作；

5.组织办理员工绩效考核工作，审查考核、培训结果；

6.员工关系工作，对员工入职、晋升、调动、离职进行面谈，了解员工思想动态；

7.为公司员工进行职业生涯规划设计；

8.向公司高层决策者提供有关人力资源战略、组织建设等方面的建议，并致力于提高公司的综合管理水平；

9.维护、发展和传播企业文化；

10.代表公司与政府对口部门、各高校和有关社会团体洽谈合作。

任职资格:

1.全日制统招本科以上学历，人力资源、管理或相关专业；

2.5年以上传媒企业相关工作经验，3年以上人力资源总监或人力资源部经理工作经验；

3.培训经历:受过战略管理、战略人力资源管理、组织变革管理、管理能力开发等方面的培训；

4.对现代企业人力资源管理模式有系统的了解和实践经验积累，对人力资源管理各个职能模块均有较深入的认识，能够指导各个职能模块的工作；

5.熟悉国家、地区及企业关于劳动合同管理、薪金制度、用人机制、保险福利待遇、培训等方面的法律法规及政策；

6.良好的服务意识、清晰的逻辑思维敏和极强的沟通能力；

7.具有一定的稳定性，能同企业共同发展。

网站开发

职位要求：
        1、学历不限,具有1年以上PHP+Mysql开发经验,精通掌握php,mysql。熟悉ajax/javascript/div+CSS，有项目经验或个人作品。
        2、拥有良好的代码习惯,要求结构清晰,命名规范,逻辑性强,代码冗余率低,注重效率、安全
        3、熟悉UCenter/Discuz!/UCenter Home者优先,做过二次开发与自主开发过应用产品者优先
        4、具有较强的团队意识，高度的责任感，对工作积极严谨，勇于承担压力
        5、热爱技术工作，喜欢钻研，执行力强。

注：本职位并非开发（维护）www.zhaotai.net，本职位人员参加公司最新互联网运营产品项目

网站编辑

岗位职责:
        1. 负责网站的内容管理和话题制造维护监测，在网站内部与网友互动；
        2. 通过内容管理帮助网站形成独特文化，并加以引导；
        3. 对相关站内活动进行执行维护；
        4. 通过网络对内容进行收集、分类、编辑、审核；
        5. 负责公司网站的内容编辑，负责板块相关栏目资料和信息的搜集、编辑、审校、更新；
        6. 针对网络热点问题撰写评论，发布相应内容文正，制造站内舆论。
       

 任职资格：:
         1. 本科学历，特别优秀者可放宽学历要求；

  2. 两年以上网站编辑工作经验；
          

3. 文字功底扎实，耐心、细致；
       

4. 对社会（网络话题）事实敏感，对网络事件、社会实践有独特见解；
 

5. 对国家互联网舆论政策有了解，并能正确把握；
 

6. 常规OFFICE 工具使用熟练，有PS功底优先；
 

7. 对互联网熟悉，3年以上网龄，有过3个以上BBS及社区账号。

执行助理

岗位职责：

1、协助公司项目策划活动、市场推广活动及公司内部大型活动的策划与组织实施；
 

2、协助市场活动文案的策划、创意、撰写和执行；
 

3、统计各项市场活动的数据，经过分类整理和分析、上报，为领导决策提供依据；
 

4、收集策划活动的市场反馈信息，进行分析并调整活动方案，建立活动模板和流程体系；
 

5、协助活动项目的推广等；
 

任职资格：

1.大学本科以上学历，专业不限；

2.在校期间热衷于各种社团活动，具备丰富的校园推广活动策划及实施经验，面试时需提供案例；

3.良好的创意思维、沟通协调能力，优秀的书面写作能力，PPT、WORD使用娴熟；

4.兴趣广泛，心态开放，乐于分享，善于合作。

  （本岗位仅限西安地区户口）

平面设计师

岗位职责：

1.《青锐·FUTURE》杂志的视觉效果设计、杂志排版；

2.公司宣传品的设计制作；

3.客户平面广告的设计制作。

任职资格：

1.美术、艺术设计、广告设计相关专业，本科学历，特别优秀者可不受学历限制；

2.精通平面设计软件Photoshop、Illustrator、Indesign、Pagemaker、Coredraw等，具备较强手绘能力者优先；

3.良好的美术功底及艺术直觉，思维开阔，创意独特，关注国际设计潮流.

4.善于沟通，学习能力强，良好的团队合作精神，较强的执行能力，能承受一定的工作强度和工作压力。

  （本岗位仅限西安地区户口）

三维设计师

岗位职责：

1.协助组织内部创作会议，激发创作思路和灵感；
 

2.负责重要的视觉设计工作（平面或三维）的具体执行；
 

3.负责创意与设计的概念衔接；
 

4.其他上级领导交办事宜。

任职资格：

  1. 美术专业专科及以上学历，1-3年工作经验
  2. 熟悉人物和道具设计，高精度人物和道具建模及贴图制作
  3. 熟悉三维游戏场景的建模和贴图制作游戏场景数据。

  （本岗位仅限西安地区户口）
